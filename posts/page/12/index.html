Hugo の始め方

Hugoのインストール Ubuntuでのインストール sudo apt-get install hugo 2. サイトの作成 hugo new site sitename sitename は自分の好きな名前に置き換えてください。 3. テーマの追加 Hugo のテーマ から好きなテンプレートを選択。 PaperMod を選択したので、 インストラクションに沿ってgit clone git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1 4. コンテンツの追加 hugo new posts/my-first-post.md --- title: "My First Post" date: 2019-03-26T08:47:11+01:00 draft: true --- ※draft: false に変更で公開 5.
