python での JSON の扱いについてわすれるのでメモ。 json.dumps json.dumps()に入れた変数を JSON string に変換する。 >>> import json >>> data = {'abc': 123} >>> type(data_json) <class 'str'> >>> data_json = json.dumps(data) >>> data_json '{"abc": 123}' json.loads json.loads(JSON string) で python dictionary に型を変換する。 >>> import json >>> data = {'abc': 123} >>> type(data_json) <class 'str'> >>> data_json = json.dumps(data) >>> data_json '{"abc": 123}' >>> load_json = json.loads(data_json) >>> load_json {'abc': 123} >>> type(load_json) <class 'dict'> >>> load_json['abc'] 123
